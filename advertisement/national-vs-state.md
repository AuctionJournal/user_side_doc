# What is the difference between national and state advertisements?

When you buy an ad package, you choose whether it targets **the whole country (national)** or **one state (state)**. That controls where on the public site your promotion can appear and whether you must pick a state in the wizard.

---

## National advertisements

**Best for:** reaching visitors across Auction Journal, not tied to one state’s listing pages.

Examples:

- **Featured Listing (National)**
- **National Print**, **National Poster**, **National Banner**
- **Blogs Ad** and **Video Ad** (promote your content nationally)

You do **not** choose a state in the flow.

---

## State advertisements

**Best for:** focusing spend where you run auctions in a specific state.

Examples:

- **Featured Listing (State)** — pick the listing, then **choose the state**
- **State Poster Ad** and **State Banner Ad** — upload creative, then **choose the state**

State ads show in **that state’s** context on the public site (see [public placement](public-placement.md)).

---

## Listing ticker (special case)

**Ticker Symbol** promotes a listing message in the **site header ticker**. It is a listing boost, not the same as “featured listing state/national” slots. See [Listing ads](listing-ads.md).

---

## How to tell in the dashboard

Package names include **National** or **State**. State packages add a **Select state** step after you set run dates.
