# Where does each advertisement appear on the public website?

Paid advertisements show on the **public Auction Journal website** (auctionjournal.com) — the site bidders and visitors browse. They do **not** appear only inside your auctioneer dashboard.

After your ad is **approved** and **live** for your selected dates, visitors may see it in featured areas that match the ad type you bought. Exact layout can vary by page; a detailed placement guide per screen will be added when available.

---

## Quick reference by ad type

| Ad type | Where visitors typically see it |
|---------|----------------------------------|
| **Ticker Symbol** | Message strip in the **site header** (scrolling ticker) |
| **Featured Listing (National)** | **National** featured listing areas (e.g. home and broad listing discovery) |
| **Featured Listing (State)** | **State** listing views for the state you selected |
| **National Print Ad** | **Print-style** promotional slot on national pages (e.g. home) |
| **National Poster / Banner** | **Image** placements on national pages |
| **State Poster / Banner** | **Image** placements on that **state’s** listing-related views |
| **Blogs Ad** | **Featured blog** areas (national promotion) |
| **Video Ad** | **Featured video** areas (national promotion) |
