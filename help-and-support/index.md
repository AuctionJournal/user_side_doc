# Help and Support

When you have a question or need help with Auction Journal, use the options below. You must be **signed in** as an auctioneer or bidder.

## Where to start

| If you are… | Open support from… |
|-------------|-------------------|
| **Auctioneer** | Sidebar **Support** → **GET HELP** |
| **Bidder** | Sidebar **Bid Support** |

You can also use **FAQs** (auctioneers: **Support** → **FAQ**; bidders: **FAQs** in the sidebar) for common answers without opening a ticket.

## Ways to get help

| Option | Best for | Guide |
|--------|----------|--------|
| **Request a Callback!** | Scheduling a phone call at a time you choose | [Callback support](./callback-support.md) |
| **Write to Us** | Detailed issues, attachments, email-style follow-up | [Email support](./email-support.md) |
| **Ticket History** | Seeing past and open requests | [Ticket history](./ticket-history.md) |
| **Chat** (floating icon) | Quick questions while you use the site | [Live chat](./live-chat.md) |
