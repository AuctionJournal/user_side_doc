# How do I get help from Auction Journal when I have a question?

Use the channel that fits your situation. All ticket-based options require you to be **signed in**.

---

## Open the support area

| Role | Steps |
|------|--------|
| **Auctioneer** | In the dashboard sidebar, open **Support** → **GET HELP**. |
| **Bidder** | In the bidder dashboard sidebar, select **Bid Support**. |

You land on a page with the heading **GET IN TOUCH** and three main choices (see screenshot below).

![GET IN TOUCH — Request a Callback, Write to Us, Ticket History](../image/help-and-support/get-in-touch.png "GET IN TOUCH")

---

## Choose the right option

### Read FAQs first (no ticket)

- **Auctioneers:** **Support** → **FAQ** for common auctioneer questions.
- **Bidders:** **FAQs** in the sidebar (separate from Bid Support).

Use FAQs when you only need general guidance.

### Auctioneers only: Listing Policy

Under **Support** → **Listing Policy** for listing rules. This is not a support ticket.

### Request a Callback!

Choose this when you want Auction Journal to **call you** at a **date and time you pick**. You get a ticket ID starting with **RAC_** and can track it under **Ticket History**. See [Callback support](./callback-support.md).

### Write to Us

Choose this for a **written request** you can follow up on: priority, message, and optional attachments. Ticket IDs start with **WTUS_**. See [Email support](./email-support.md).

### Ticket History

Use this to see **open and past** callback and email requests. See [Ticket history](./ticket-history.md).

### Chat icon (bottom of the screen)

A **floating chat button** (speech bubbles) is available on many pages for **quick questions**. It does not replace tickets when you need a tracked case. See [Live chat](./live-chat.md).

---

## Quick comparison

| Need | Use |
|------|-----|
| General how-to | **FAQ** (and **Listing Policy** for auctioneers) |
| Phone call at a set time | **Request a Callback!** |
| Detailed issue with files and replies | **Write to Us** |
| Check status of a request | **Ticket History** |
| Immediate short question | **Chat** icon |
