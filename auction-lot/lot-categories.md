# Explain lot categories

Last modified: 2026-05-28

Each lot belongs to a category in Auction Journal.

Auctioneers set the category when creating a lot, and categories are also used during lot import and lot search.

---

## How category works while creating a lot

- In **New Lot**, auctioneer selects a category for that lot.
- Category helps bidders discover similar lots faster.

---

## Category in Import LOTS (CSV)

For import, Auction Journal accepts **category ID**, not category name.

- Use the **Category** column with a numeric ID.
- Category name text is not accepted in that column.
- Category ID should be a **positive number**.

Example:
- `12` (valid style)
- `ANTIQUES` (not valid in import category field)

---

## How users use categories

Users can search/filter lots by category on Auction Journal to find relevant lots quickly.

---
