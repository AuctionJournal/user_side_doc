# How to manage auction images while creating auction

Last modified: 2026-05-28

You can manage auction images during auction build, but only after the auction has an **Auction ID**.

---

## Important rule first

You must create/save the auction first so it gets an **Auction ID**.  
Before that, image upload is disabled and you see a warning to create Auction ID.

---

## What you can do with auction images

After Auction ID is created, open **Auction Images** / **Auction Preview Gallery** and you can:

- add one or multiple images
- move image position/order
- crop an image
- delete an image
- preview full image

---

## How image position works

Use the **Move To** field/action on each image card to change its position in gallery order.

This order controls featured images shown first in auction presentation.

---

## How cropping works

Use the **crop** icon on an image card.

- Crop opens for that specific image
- Save replaces that same image with the cropped version
- Repeat for any other image as needed

---

## Suggested workflow

1. Create auction draft (get Auction ID)
2. Upload all base images
3. Reorder with **Move To**
4. Crop where needed
5. Save draft / publish when ready
