# How do I see my registered listings?

After you [register for a listing](register-for-listing.md) on the public site, your registration appears in the **Bidder Dashboard**. Where you look depends on **how** you registered:

| How you registered on the listing page | Where to view it in the dashboard |
|----------------------------------------|-----------------------------------|
| **GENERATE BID PASS** ( **OnSite Auction** listings) | **My Passes** |
| **REQUEST A CALLBACK** (all other listing types) | **My Requests** |

You must be **signed in as a bidder**. These pages are not on the public marketing site—they are under `/bidder/…` after login.

---

## Open My Passes (bid passes)

Use this for listings where you selected **GENERATE BID PASS**.

1. Sign in to the **Bidder Dashboard** ([how to use the dashboard](../bidder/dashboard.md)).
2. In the left menu, select **My Passes**.

   Direct link when already signed in: [auctionjournal.com/bidder/my-passes](https://auctionjournal.com/bidder/my-passes)

3. The page heading is **My Passes**. You see a card for each listing where you generated a bid pass.

If you have none yet, the page shows: **You don't have any bid passes yet.**

---

## Open My Requests (callback requests)

Use this for listings where you selected **REQUEST A CALLBACK** (live webcast, online timed, online absolute, live webcast with onsite, and similar).

1. Sign in to the **Bidder Dashboard**.
2. In the left menu, select **My Requests**.

   Direct link when already signed in: [auctionjournal.com/bidder/my-requests](https://auctionjournal.com/bidder/my-requests)

3. The page heading is **My Requests**. You see a card for each listing where you asked for a callback.

If you have none yet, the page shows: **You haven't made any requests yet.**

---

## What each card shows

On **My Passes** and **My Requests**, each entry uses the same card layout:

| Field | Meaning |
|-------|---------|
| Photo | First image from the listing |
| Company name | Auctioneer’s **Company Name** |
| **PRODUCT NAME** | Listing **auction title** |
| **ADDRESS** | Listing location |
| **DATE** / **TIME** | Scheduled auction date and time |
| **VIEW AUCTIONEER DETAILS** | Opens that auctioneer’s public profile on Auction Journal |

These pages are for **listing** registrations (bid pass or callback). They are **not** the same as **Auction Details** in the sidebar, which is for **auctions** you registered to bid in.

---

## Quick checklist

```text
OnSite listing → GENERATE BID PASS → My Passes
Other listing types → REQUEST A CALLBACK → My Requests
```

- You can register on a listing only **once** per account; it then stays in the matching Passes or Requests list until the auctioneer or data retention rules remove it from your view.
- Auction Journal does **not** currently let you cancel a pass or callback from these pages—you view what you already submitted.
